1 Project Overview

SecuAuth AI is an invisible continuous authentication layer designed for the Indian securities market. It adds a silent biometric + cryptographic verification system that activates after normal login (Password + OTP). Even if a hacker steals the credentials, the system verifies that the real investor is physically present using the device camera — with zero extra steps for the legitimate user.

2 Architecture & Workflow Diagram

Workflow Explanation:
The diagram shows the complete flow: User enters Password + OTP → Silent camera scan activates → Facial landmarks are converted into a cryptographic signature → Backend verifies the signature → Final decision (Access granted or Intruder alert). The entire process completes in under 1.5 seconds.

3 The Problem We Are Solving

90%+ of account takeovers happen through stolen credentials. Even with OTP, hackers use real-time phishing kits to intercept the OTP. Once the correct OTP is entered on the legitimate site, the system grants full access. Existing biometric solutions (Face ID, fingerprint) are visible and can be bypassed using backup PINs.

7 Privacy & Security

No Biometric Data Sent

Only cryptographic signatures are sent. No face image is transmitted.

Keys Stored on Device

Private keys stay in Secure Enclave. Server only has public keys.

8 Key Advantages

Zero Friction UX

No extra steps for legitimate users. Everything runs in background.

Intruder Capture

System captures photo evidence of the hacker automatically.

9 Technologies We Will Use

Frontend (Chrome Extension)

MediaPipe Face Mesh — Real-time facial landmark detection (468 points)
WebCrypto API — Cryptographic key generation and signing
Manifest V3 — Modern Chrome Extension framework
Content Scripts — Inject into broker login pages

Mobile SDK

Android: Kotlin + MediaPipe + Android Keystore
iOS: Swift + Vision Framework + Secure Enclave

Backend

Node.js — Signature verification server
ECDSA / RSA-4096 — Asymmetric cryptography for verification
Express.js — Lightweight API server
